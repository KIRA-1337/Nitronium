# Info Mod для Minecraft 1.21.1 Forge

Мод отображает информацию об игроке в небольшом окне с настраиваемым положением.

## Функции

- **Координаты** - текущие X, Y, Z позиции игрока
- **Скорость** - горизонтальная скорость движения
- **Сид мира** - seed текущего мира
- **Ник игрока** - имя персонажа

## Требования

- Minecraft 1.21.1
- Forge 51.0.29+
- Java 21

## Сборка

```bash
# Требуется JDK 21
./gradlew build
```

Собранный файл будет в `build/libs/infomod-1.0.0.jar`

## Настройка положения

Положение информационного окна настраивается в файле:
`config/infomod-client.toml`

```toml
[overlay]
    # Позиция X (от левого края экрана)
    positionX = 10
    # Позиция Y (от верхнего края экрана)
    positionY = 10
```

## Установка

1. Установите Minecraft Forge 1.21.1
2. Скопируйте `infomod-1.0.0.jar` в папку `mods`
3. Запустите игру
